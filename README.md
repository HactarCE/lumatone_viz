# Lumatone Visualizer

## Usage

1. Download/install [Rustup](https://www.rust-lang.org/tools/install).
2. Clone this project and build/run:

```
git clone https://github.com/HactarCE/lumatone_viz
cd lumatone_viz
cargo run --release
```

On Windows, you may need an external program such as [loopMIDI](https://www.tobias-erichsen.de/software/loopmidi.html) to create a virtual MIDI device.
